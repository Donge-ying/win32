# Executables

This directory contains the executables built from the examples in the
[examples](../) directory.

To build the executables from the examples, run the following command:

```sh
tools\build.cmd
```

This will build the following executables:

- `dartnote.exe`
- `hello.exe`
- `snake.exe`
- `taskdialog.exe`
- `tetris.exe`
